# Products (Netology)

Учебный проект: реализация простого хранилища (репозитория) для товаров и менеджера для добавления и поиска товаров.

## Структура проекта

### Основные классы (`src/main/java/ru/netology`)

- **Product** — базовый товар.  
  Хранит поля `id`, `name`, `price`.

- **Book** — товар-книга (наследник `Product`).  
  Дополнительно хранит автора (например, `author`).

- **Smartphone** — товар-смартфон (наследник `Product`).  
  Дополнительно хранит производителя (например, `manufacturer`).

- **ProductRepository** — репозиторий (хранилище) товаров на основе массива `Product[]`.  
  Основные операции:
    - сохранение товара в массив
    - получение всех сохранённых товаров
    - удаление товара по `id`

- **ProductManager** — менеджер товаров.  
  Работает через `ProductRepository` (передаётся в конструктор).  
  Основные операции:
    - добавление товара в репозиторий
    - поиск товаров по строке (например, по названию)

- **FakerUtils** — утилита генерации тестовых данных (случайные `id`, названия, цены, книги, смартфоны).
